about the game

This game was created for a school project. The intent was, to create a game with a high skill ceiling. It should be a Jump'N'Run in pixel art designed. The goal is to complete the game with a high score.
The game was created in Unity3D with the coding language C# in the Editor VisualStudio 2017 Community Edition.
